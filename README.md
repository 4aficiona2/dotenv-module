## Setup
- Add `nuxtjs-dotenv-module` dependency using yarn or npm to your project
- Add `nuxtjs-dotenv-module` to `modules` section of `nuxt.config.js`

```js
{
  modules: [
    // Simple usage
    'nuxtjs-dotenv-module',

    // With options
    ['nuxtjs-dotenv-module', { /* module options */ }],
 ]
}
```
